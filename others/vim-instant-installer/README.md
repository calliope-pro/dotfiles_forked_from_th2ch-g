# vim-instant-installer

## Requirement
- vim >= 8.2 tested


## Usage
atom theme
~~~
git clone https://github.com/th2ch-g/dotfiles && \
cd dotfiles/others/vim-instant-installer && \
./install.sh --atom
~~~

vscode theme
~~~
git clone https://github.com/th2ch-g/dotfiles && \
cd dotfiles/others/vim-instant-installer && \
./install.sh --vscode
~~~
